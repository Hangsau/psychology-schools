# psychology-schools — 計畫

> 心理學學派**完整內容**資料庫（非概念、非簡介）。
> 獨立 repo，與 `religions-history` 共用「人的 13 個問題領域」對接軸。
> 啟動：2026-07-16

---

## 1. 目標

把一份只有學派名的「心理學學派總覽」框架，填成**完整、可信、可追溯**的內容庫：

- **完整**：每個學派涵蓋 歷史脈絡 / 代表人物 / 完整理論主張 / 關鍵著作書目 / 方法技術 / 批評與後續發展。不是一句話簡介。
- **可信**：每條宣稱標來源等級 + 確定性標記；三角驗證；主動比對「教科書扭曲清單」。
- **可追溯**：缺口、未驗證、爭議項目全部**顯性列出**，不藏。

**不是**：不做原典全文重製（版權），不做圖表 / 看圖 / 製圖類任務（生不出來的不硬塞）。

---

## 2. 與 religions-history 的對接（核心）

religions-history 已確立一根軸：**人從生到死的 13 個問題領域**（存在與意義、苦難疾病、無常老死、信仰超越……），經文是「歷代對這些問題的回答」，心理學是**解讀透鏡**。

本專案補的正是那面透鏡的知識源。對接靠**共同軸**，不靠文本形式：

- 每個學派標 `domain_tags[]` ← religions-history 的 13 領域（見 `00-overview/crosswalk-13-domains.md`）
- meta.json 欄位與 religions-history 對齊，索引格式一致，未來可直接 merge
- 結合效果：一個問題（如「死亡」）能同時拉出——歷代經文怎麼答 + 各心理學派怎麼答

**本質差異**：religions-history「原文優先」（權威在原文）；本專案「綜述優先」（內容在理論體系，且原典多有版權）。

## 3. 版權與正確性（已解決的方法論）

**版權擋的是「重製」，不是「查閱」。** 驗證只需讀取權（圖書館 / 購買 / archive.org / 學術庫），不需散布權。所以「對照原始出處寫」始終可行，只是不公開全文。

三種缺陷各有解法（詳見 `methodology/verification-sop.md`）：

| 缺陷 | 解法 |
|------|------|
| 誤解（教科書傳話遊戲）| 每學派維護 known-distortions 檢查清單，比對批判性文獻 |
| 遺漏 | 完整性模板（空欄=顯性缺口）+ 對照標準《心理學史》目錄反查 |
| 事實錯誤 | 三角驗證（≥2 獨立權威一致才收），分歧→兩說並列 |

誠實邊界：能做到「可信、可追溯、缺陷透明」，非「保證無誤」。

## 5. 工作方法

- **1 學派 = 1 檔 = 1 次 m3 呼叫**（`schools/<slug>/synthesis.md`）
- 每篇強制含「已知缺口 / 未驗證 / 爭議」區
- 確定性標記 🔵確定 🟢高 🟡中 🟠低 🔴存疑
- 來源等級 A原典 / B同儕審查·SEP·學術教科書 / C百科 / D科普
- 全繁體中文，無簡體混用

---

## 6. Anti-pattern

- ❌ 硬塞生不出來的內容（圖表 / 看圖 / 製圖）→ 直接省略，不寫假的
- ❌ 引用拼裝（citation patchwork）→ 綜述吸收後自己寫
- ❌ 單一來源當定論
- ❌ 把教科書扭曲版當事實（Maslow 金字塔、旁觀者效應 38 人…）
- ❌ 捏造引用 / 假書目 → 查不到出處的一律刪
- ❌ 簡繁混用
