# 綜述驗證 SOP

> 每篇 `synthesis.md` 交付前**強制**跑過。目的：在拿不到 / 不重製原典全文的前提下，把「誤解 / 遺漏 / 事實錯誤」壓到最低，並讓殘餘風險**顯性可追溯**。

## 前提：版權擋重製，不擋查閱

驗證只需**讀取權**（圖書館 / 購買 / archive.org 借閱 / Google Books 片段 / 學術資料庫），不需散布權。所以「對照原始出處寫」始終可行——只是不把受版權全文貼進 repo。

---

## 三種缺陷 × 對應解法

### 1. 誤解（最危險）— known-distortions 檢查

二手資料最大的坑是「教科書傳話遊戲」：一個扭曲版本被反覆抄。寫每個學派時，**主動比對已知扭曲清單**（附錄 A），命中就用校正版並在 §8 註明。

### 2. 遺漏 — 完整性模板

每篇 `synthesis.md` **強制**含 8 段（見下）。空欄位寫「資料待補」，**不靜默略過**——空白＝顯性缺口。另對照標準《心理學史》目錄反查漏掉的人物/分支。

### 3. 事實錯誤 — 三角驗證

任何**具體宣稱**（年代 / 「X 主張 Y」/ 數字 / 誰影響誰）需 **≥2 個獨立權威來源一致**才收 🟢+。分歧 → 兩說並列標 🟡。查不到出處 → 刪或降 🔴。

---

## synthesis.md 固定 8 段

1. **定位與歷史脈絡** — 何時何地、回應什麼問題、承接/反對誰
2. **代表人物** — 生卒 + 關鍵貢獻
3. **核心理論主張** — 完整展開（**非一句話簡介**），關鍵術語附原文
4. **關鍵著作書目** — 書名 / 年代 / 版權標記（PD/©）
5. **方法 / 技術** — 尤其治療取向的具體做法
6. **批評與後續發展** — 學界批評、實證支持度、後續分支
7. **對接 13 領域** — 標 `domain_tags` + 一句為何（見 `crosswalk-13-domains.md`）
8. **已知缺口 / 未驗證 / 爭議** — 誠實列出本篇沒查證的、有爭議的、待補的

## 標記系統

**確定性**：🔵確定（多權威一致 + 常識）｜🟢高｜🟡中（單一好來源 / 有分歧）｜🟠低｜🔴存疑（待查/可能錯）

**來源等級**：A 原典（讀，不貼全文）｜B 同儕審查專著·SEP·學術教科書｜C 百科（Britannica/維基交叉）｜D 科普

每個具體宣稱後標來源等級，例：`（Beck 1976, A）` `（SEP "Behaviorism", B）`。

---

## 禁令

- ❌ **捏造引用 / 假書目**：查不到真實出處的一律刪或標 🔴。引用必須能對到真實文獻。
- ❌ 圖表 / 製圖 / 看圖類內容：生不出來就省略，不寫假的。
- ❌ 單一來源當定論。
- ❌ 引用拼裝：綜述吸收後自己寫，不逐句掛 source。
- ❌ 簡繁混用。

## 附錄 A：已知教科書扭曲清單（持續增補）

寫到相關學派時**主動檢查**，命中用校正版：

| 學派 | 常見扭曲 | 校正 |
|------|---------|------|
| 人本 | Maslow 畫了需求金字塔 | Maslow **從未**畫金字塔；金字塔是後人（管理學教材）加的 |
| 行為主義 | 小 Albert 被去除制約、結局圓滿 | Watson **從未**去除制約；Albert 身分與命運是後來研究才追查 |
| 行為主義 | Pavlov 用「鈴」 | 多為簡化/誤譯；Pavlov 主要用節拍器、蜂鳴器、燈光等 |
| 社會心理學 | 旁觀者效應「38 人見死不救」（Kitty Genovese）| 《紐時》原報導大幅失實；目擊人數與「無人反應」被誇大 |
| 社會心理學 | Milgram 服從實驗乾淨可信 | 後來檔案顯示有腳本外施壓、結果被選擇性呈現 |
| 生理心理學 | Phineas Gage 從溫和變全然狂暴 | 人格改變被戲劇化；後期有相當程度復原 |
| 認知/發展 | Piaget 階段是嚴格年齡界線 | 階段更連續、有文化與任務差異 |
| 精神分析 | 冰山「本我/自我/超我」是 Freud 原圖 | 冰山比喻多為後人視覺化，非 Freud 精確原意 |

> 命中以上任一，§8 註明「已避開 XX 常見扭曲」。發現新扭曲 → 增補本表。
